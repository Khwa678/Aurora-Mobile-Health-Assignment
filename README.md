# 🚀 Aurora Mobile Health – AI-Powered Healthcare Assistant

## 📌 Overview

Aurora Mobile Health is an AI-powered healthcare application designed to provide users with intelligent health assistance, personalized guidance, and seamless access to healthcare-related information. The platform leverages Google's Gemini AI capabilities to deliver conversational support, health insights, and an enhanced digital healthcare experience.

Built with modern web technologies, Aurora Mobile Health demonstrates the integration of Artificial Intelligence into healthcare solutions while maintaining a user-friendly and responsive interface. The project showcases how AI can be utilized to improve accessibility, engagement, and information delivery within the healthcare domain.

---

## 🎯 Project Objectives

The primary goal of Aurora Mobile Health is to bridge the gap between users and healthcare information by providing:

* AI-powered conversational assistance.
* Easy access to healthcare-related guidance.

* Secure handling of user interactions and API communication.

This project serves as a practical implementation of modern AI technologies in healthcare-focused applications.

## 🏗️ Project Architecture

Aurora Mobile Health follows a modern client-server architecture where:

1. Users interact with the web interface.
2. Requests are processed through the application server.
3. Gemini AI handles intelligent response generation.
4. Responses are delivered back to the user in real time.
5. Environment variables securely manage API credentials.

This architecture ensures scalability, maintainability, and efficient AI integration.

## 🚀 Installation and Setup

### Prerequisites

Before running the project, ensure you have:

* Node.js installed
* npm installed
* A valid Gemini API Key

### Clone the Repository

```bash
git clone https://github.com/your-username/aurora-mobile-health.git
cd aurora-mobile-health
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file in the root directory and add:

```env
GEMINI_API_KEY=your_api_key_here
```

### Run the Application

```bash
npm run dev
```

The application will start locally and be accessible through your browser.
